Explainable Audio Emotion Recognition Using High-Resolution Autoencoder and Prototype-Based Deep Learning

This webpage complements the paper "Explainable Audio Emotion Recognition Using High-Resolution Autoencoder and Prototype-Based Deep Learning". It provides the learned prototypes in both image and reproducible audio for the three different datasets studied.

The accompanying repository contains all source code necessary to conduct the experiments described in the paper, as well as the figures that appear in it. You can explore the project on GitHub:

View on GitHub

This page is structured in foldable sections: for each dataset, two male and two male speakers are explained: we reproduce their top 10 prototypes per emotion class according to the prototype scoring metric detailed in the paper.
